# Flashcard App

A simple flashcard web app for studying terms and definitions. Built with vanilla JavaScript and Bootstrap 5.

## Features
- Choose from available flashcard collections
- Flip card animation for term/description
- Track correct/incorrect answers per term (saved in localStorage)
- Switch between "Term First" and "Description First" modes
- View overall correctness percentage

## Usage
Open the app in your browser or visit the published version:

👉 [https://pamblam.github.io/flashcards/](https://pamblam.github.io/flashcards/)

## Customizing
- Add new datasets as JSON files in the `datasets/` folder
- Edit `main.js` to add new dataset entries if needed
